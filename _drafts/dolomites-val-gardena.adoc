= 多洛米蒂的加尔代纳山谷 (Val Gardena, Dolomites)
:page-liquid: true
:toc: macro
:toclevels: 2
:sectnums:
:imagesdir: https://assets.lilyroger.com/images/trips/2026/dolomites/dolomites-val-gardena

如果多洛米蒂 (Dolomites) 是上帝留在人间的巨作，加尔代纳山谷 (https://www.valgardena.it/en/[Val Gardena]) 就是这本巨作的封面。北面有刀锋山 (Seceda) 的震撼断崖，南面有休斯高原 (Alpe di Siusi) 的童话草甸，再加上拉定文化 (Ladin) 的古老底蕴。在这篇笔记中，我们收集了加尔代纳山谷的核心信息，开始认识这座多洛米蒂最知名的山谷。

[[ladinia]]
== 拉定尼亚 (Ladinia)

拉定人 (Ladins) 是分布于意大利北部的一个民族。多洛米蒂的五个古老拉定山谷共同构成了文化和地理上的**拉定尼亚** (Ladinia) 大区。

* 加尔代纳山谷 (Val Gardena)
* 巴迪亚山谷 (Val Badia)
* 利维纳隆戈山谷 (Livinallongo)
* 法萨山谷 (Val di Fassa)
* 安佩佐山谷 (Val d'Ampezzo)

加尔代纳山谷是**拉定五谷**之一，而且是其中的经济和文化的心脏。

[[western-dolomites]]
== 西多洛米蒂 (Western Dolomites)

多洛米蒂可以划分为**西多洛米蒂**和**东多洛米蒂**两部分。连接巴迪亚山谷中的**科尔瓦拉镇** (Corvara) 和利维纳隆戈山谷中的**阿拉巴镇** (Arabba) 的**坎波隆戈山口** (Campolongo Pass)，正好坐落于东、西多洛米蒂天然地理分界线上。

加尔代纳山谷是西多洛米蒂的核心，与东多洛米蒂的核心安佩佐山谷通常被称为多洛米蒂的“双子星”。第一次去的旅行者，最经典的走法就是在这两个核心山谷各住几天，即可完美打包多洛米蒂 80% 的绝美风光。

[[puez-odle]]
== 奥德尔群峰与普埃兹群峰 (Puez-Odle)

加尔代纳山谷的北侧是**奥德尔群峰** (Odle Group) 和**普埃兹群峰** (Puez Group)。

**奥德尔** (Odle) 意大利语和拉定语，字面意思是“针”。拉定先民们从奥德尔群峰的另一侧的富内斯山谷 (Val di Funes) 中看奥德尔群峰，这排严重风化的山峰就像高耸入云的“针”，因而得名。奥德尔群峰的德语名字**盖斯勒群峰** (Geisler Group) 也很常见。

在奥德尔群峰以东，是**普埃兹群峰** (Puez Group)。两组群峰共同构成了普埃兹-奥德尔自然公园 (Parco Naturale Puez Odle)。

奥德尔群峰有着刀锋山绝美断崖，和富内斯山谷中的孤独小教堂，是多洛米蒂的经典名片，全世界游客趋之若鹜。

[[alpe-di-siusi]]
== 休斯高原 (Alpe di Siusi)

休斯高原是一个巨大的高山台地，像一个平坦的空中花园，凌空高悬在加尔代纳山谷的南侧峭壁之上，是欧洲最大的高山草甸。同样使用率很高的 Seiser Alm 是休斯高原的德语名称。

站在加尔代纳山谷中央，向北看是凌厉尖锐的刀锋山，向南看则是温柔起伏的休斯高原。两处景观一刚一柔，构成了西多洛米蒂的标志性双翼。

[[sassolungo-group]]
== 长石群峰 (Sassolungo Group)

无论你身处加尔代纳山谷哪里，只要抬头向东南边的山谷尽头望去，都能看到加尔代纳山谷地标，雄伟的**长石群峰**。

长石群峰是由几座白云石山峰组成的马蹄形的群峰。其中最著名的三个山峰是（中文名来自意大利语名字的字面意思）：

* 长石山 (Sassolungo): 长石群峰的最高峰。
* 平石山 (Sassopiatto): 位于群峰的西侧，一侧是巨大平滑岩坡，可以徒步登顶。
* 五指山 (Cinque Dita): 夹在主峰和平石山之间，顶级的攀岩圣地。

长石山和五指山之间，有一个巨大的 V 字形鞍部，被称为**长石山垭口** (Forcella del Sassolungo)。从塞拉山口 (Sella Pass) 出发，你可以乘坐标志性的白色复古双人站立式缆车，直达垭口之上的 Toni Demetz 山屋。这个缆车因为其“别致的“的外形，常被戏称为“棺材缆车”。

围绕整座群峰还有经典的长石山大环线徒步路线 (Sassolungo Circuit)。全长约 16-17 公里，累计爬升约 800-900 米，完整走完需要 5.5 到 6.5 小时。是多洛米蒂的顶级徒步路线之一。

[[sella-group]]
== 塞拉群峰 (Sella Group)

**塞拉群峰**是多洛米蒂山区中地位最高的中央核心山块。从谷底往上看，塞拉群峰是高耸云霄的白云岩岩壁，犹如一座要塞。但如果乘坐缆车登上它的顶端，眼前却是一片极其开阔、寸草不生、布满碎石的银灰色荒漠台地，常被称为“地球上的月球表面”。

塞拉群峰被四大著名的公路山口环绕，这四个山口无缝串联起了拉定五谷中的四个：

* 加尔代纳山口 (Gardena Pass): 北侧，连通加尔代纳山谷和巴迪亚山谷
* 坎波隆戈山口 (Campolongo Pass): 东侧，连通巴迪亚山谷和利维纳隆戈山谷
* 波尔多伊山口 (Pordoi Pass): 南侧，连通利维纳隆戈山谷和法萨山谷
* 塞拉山口 (Sella Pass): 西侧，连通法萨山谷和加尔代纳山谷

其中，加尔代纳山谷由西向东延伸 20 多公里之后，在塞拉群峰的脚下一分为二：一条路翻过塞拉山口翻山前往法萨山谷中加纳泽伊 (Canazei)；另一条翻过加尔代纳山口前往巴迪亚山谷中的科尔瓦拉 (Corvara)。

此外，从波尔多伊山口搭乘快速大缆车仅需 4 分钟就可以直接上到塞拉群峰的顶端，号称“多洛米蒂露台” (Terrace of the Dolomites) 的**萨斯波尔多伊** (Sass Pordoi) 观景台。

[[sellaronda]]
== 塞拉隆达 (Sellaronda)

**塞拉隆达**是多洛米蒂山区最著名的环形景观路线。它环绕着雄伟的塞拉群峰，在冬季是阿尔卑斯山最壮丽的滑雪环线，在夏季则是多洛米蒂自驾和单车骑行的最美路线。

塞拉隆达环线横跨拉定山谷中的四个(安佩佐山谷在塞拉群峰的最东边，不在塞拉隆达环线上)，加尔代纳山谷正是其中之一。

* 加尔代纳山谷: 塞拉群峰的西北侧
* 巴迪亚山谷: 塞拉群峰的东北侧
* 利维纳隆戈山谷: 塞拉群峰的东南侧
* 法萨山谷: 塞拉群峰的东南侧

[[three-core-villages]]
== 加尔代纳三镇

加尔代纳山谷的底部由三个风情各异的阿尔卑斯小镇组成。从西部的谷口向东部的谷底依次排开。

[[ortisei]]
=== 奥尔蒂塞伊 (Ortisei)

**奥尔蒂塞伊**是加尔代纳山谷中面积最大、最繁华的中心小镇。镇上拥有多洛米蒂地区最精致的彩色历史步行街，沿街开满了传统的意式咖啡馆、精品店和拉定手工艺木雕工坊。

奥尔蒂塞伊是整个西多洛米蒂地区的最佳旅行大本营。住在这里，完全不需要开车，步行就能从镇中心搭乘缆车直达三大高山台地：

* 刀锋山 (Seceda) 缆车站（向北）
* 休斯高原 (Alpe di Siusi) 缆车站（向南）
* 拉斯切萨高原 (Resciesa) 缆车站（向西北）

[[santa-cristina]]
=== 圣克里斯蒂娜 (Santa Cristina)

**圣克里斯蒂娜**位于加尔代纳山谷的正中央，规模是三个小镇中最小、最安静的一个，保留了更多传统阿尔卑斯农场的气氛。

圣克里斯蒂娜直面雄伟的长石群峰马蹄形的的正面。随时随地有着绝美的山景。镇上的 Col Raiser 缆车站是前往盖斯勒群峰下方高山草甸 (Col Raiser) 以及刀锋山后山的绝佳徒步起点，是奥尔蒂塞伊的刀锋山缆车的完美组合和备用方案。

因为夹在两个大镇之间，圣克里斯蒂娜的酒店性价比通常最高。

[[selva]]
=== 塞尔瓦 (Selva)。

**塞尔瓦**位于加尔代纳山谷的最东端，也是最高的小镇，海拔约 1563 米，直接依偎在巍峨的塞拉山群和长石群峰的脚下。因为小镇直接坐落在**塞拉隆达环线**上，作为塞拉隆达的最佳门户，冬季是全球知名的滑雪胜地，夏季是自驾和单车骑行的最佳大本营。
